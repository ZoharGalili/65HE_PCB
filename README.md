# 65HE_PCB

this is a 65% hall effect keyboard pcb running [libhmk](https://github.com/peppapighs/libhmk) firmware.

## Board Renders

[front](renders/front.jpg)
[back](renders/back.jpg)

## Features

- all the usual features of an HE keyboard listed in the [libhmk](https://github.com/peppapighs/libhmk) repo.
- multilayout support (7u/6.25u spacebar, split backspace, stepped caps).
- bakeneko 65 combined with ai03 vega board outline supports gummy o-ring mount.
- at32f405 microcontroller enables 8khz polling rate.
- uses 4 of the ai03 vega outer plate screws.
